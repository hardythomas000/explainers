CNC Programmer Operating Manual
Exactly what to press, where it is, and what it does.
1Start Here - What To Press First
Order Press Result
Step 1 Load G-code (.nc / .eia) Parses toolpaths, tool numbers, operations, RPM/feed/coolant, max Z, NC filename, revision, and part hints.
Step 2 Load Tool Library Loads tool descriptions, holder, diameters, stickout, and tool images from HTML/CSV/.tooldb + image files.
Step 3 Machine picker then choose MAZAK or HAAS Applies machine-specific rules (including Mazak Z2 behavior).
Step 4 Edit fields directly on sheet (NC/Part/Revision/Material/Programmer/Datum/Notes) Final operator-facing setup instructions are locked in.
Step 5 Print Z2 Tools Print Tools Only Print Setup Only Export Excel Export HTML Exports exactly the view needed for machinist/shop floor handoff.

3Screen Areas - What Each Section Is For

1) Workholding & Datum

  • Fixture/setup instructions for the operator.
  • Stock dimensions and datum origin definitions.
  • Must be correct before release.

2) Notes

  • Short, high-value notes only.
  • Use highlight colors for safety/quality critical points.

3) Setup Images

  • Visual orientation and fixture evidence.
  • Paste screenshots with Ctrl+V or Win+V.

4) Toolpaths

  • Operation sequence and cutting parameters.
  • Validate tool, RPM, feed, max Z, coolant.
4Common CNC Programmer Flows

Flow A - New Job

Load NC file
Load tool library + images
Confirm machine, part, revision
Confirm toolpath values
Add setup notes/images

Flow B - Mazak Check

Confirm part number (example: 26A33476)
Confirm NC filename separately
Confirm tools above T124 marked Z2
Use Print Z2 Tools output if required
